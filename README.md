## 기능 구현 순서

### 1. 랜덤 숫자 배열 생성
 1 ~ 9 까지 서로 다른 수를 랜덤하게 생성해서 배열에 담는다. 
 
### 2. 사용자 입력 배열 생성
 `Scanner`로 사용자에게 3자리 수를 받아 배열에 담는다. 
>**exception**
> 
>숫자를 입력하지 않은 경우
> 
> 숫자의 개수가 3개가 아닌 경우
> 
> 숫자 중 0이 있는 경우
> 
> **guard**
> 
> 무효로 하고 다시 입력

### 3. 스트라이크 확인

인덱스와 값이 모두 같으면 `strike++;`

### 4. 볼 확인

값만 같으면 `ball++;`

### 5. 결과 출력
* 성공(`strike == 3`)

```
3 스트라이크 아웃
3개의 숫자를 모두 맞히셨습니다! 당신의 승리입니다.
게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
```
출력 후 계속할지 인풋을 받는다. 

* 실패(`strike != 3`)

```
x 스트라이크 y 볼
둘다 없으면 
볼넷
```
